# server-client-model
这是我写的一个cs(客户机-服务器)架构的demo，是一个简洁、轻巧的Hello World级别的项目。
# 运行顺序
在main包里有两个java类，一个是服务器启动类，另一个是客户端启动类。先运行服务器启动类，再运行客户端启动类。然后在客户端的控制台输入命令，则在服务器控制台有相应的相应。例如：当在客户端控制台输入：现在几点了，则在服务器控制台输出相应时间；如果输入：你叫什么名字，则输出Weebing。
